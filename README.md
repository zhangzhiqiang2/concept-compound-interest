# 概念复利 · Concept Compound Interest

> 每天理解一个概念，存入大脑，在做事时调用它，让生活变好。

## 这是什么

这是一个"每日一概念"的个人知识积累计划。

每天花时间深入理解一个概念：查资料、找案例、验证它对不对、为什么对、怎么用。把理解写下来，存入大脑。以后每次做决策、遇到问题时，大脑里就有了越来越多的"思维武器"。

## 为什么叫"概念复利"

每天学一个概念，当天看可能没什么感觉。但坚持一年就是 365 个概念。这些概念之间还会连接、碰撞、生发出新的理解。这就是认知上的复利效应——和投资一样，时间越长，威力越大。

## 规则

- **每天**理解一个概念
- **时间**：每天上午 8:30
- **形式**：写一篇笔记，包含定义、论据、应用、个人思考
- **上传**：每天推送到 GitHub
- **核心**：不是为了"记下来"，而是为了"真正理解、能调用"
